### 結構細節解析

| 符號 | 描述 | 目的 |
| :--- | :--- | :--- |
| `^` | **行首錨點** | 確保匹配從行首開始。 |
| `#` | **字面量** | 匹配**恰好一個** `#` 字符。 |
| `\s` | **空白字符** | 匹配 `#` 後面必需的一個空格。 |
| `.` | **任意字符** | 匹配標題文字的任一字符。 |
| `*` | **量詞 (零次或多次)** | 匹配標題文字（`# ` 後的所有內容），零個或多個。 |

### 💡 程式碼中的應用

如果您將這個模式用於 `highlightBlock` 中的高優先級檢查，它將完全涵蓋所有 H1 標題行：

```python
# 僅匹配 H1 標題
self.pattern_header_h1 = QRegExp(r"^#\s.*") 
```
